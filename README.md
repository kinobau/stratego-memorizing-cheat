# STRATEGO.COM MEMORIZING CHEAT

Simple console script that makes opponent pieces on the stratego board stay visible for longer than intended.

**Why?**
Because remembering 40 hidden pieces is hard in this generation.

**How to use**
1. Open the developer console when you load into a game
2. Paste the script into the console and press enter.
3. Attack or get attacked and never forget what is what.

100% client-side, no server communication.
